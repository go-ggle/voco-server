# VOCO
>  캡스톤디자인과창업프로젝트 14팀 고글 <br>
>  프로젝트 기간 2022.09 - 2023.06

<br>

## 🎙 서비스 소개
VOCO는 콘텐츠 제작자의 해외 시청자 유입을 돕기 위해 음성 변환 기술을 이용하여 사용자의 목소리로 만들어진 외국어 더빙을 제공합니다.
